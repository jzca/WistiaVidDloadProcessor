# WistiaVidDloadProcessor
JS function of processing to download Wistia Videos

## How to download Wistia Videos

1. Inside a video, Right-click and click 'Copy link and thumbnail'
2. Put the raw data into a variable
3. Put variables into the array
4. Use Console to run my codes

5. You will see the link(s)
6. Drop them into https://getvideo.at/en/
7. Choose the first 480p, if there are two of them and you want higher quality of the video. 480p could be 1080p or etc
